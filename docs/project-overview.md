# Vantage Finance - Project Overview

## Project Mission
A precision debt and wealth management application specifically engineered for "Charith & Family." The platform provides tools for tracking income, expenses, budgets, debts, and savings goals with a focus on strategic financial planning.

## Technology Stack

### Frontend
- **Framework**: React 19 (Vite)
- **Styling**: Tailwind CSS
- **State/Data Fetching**: TanStack Query (React Query)
- **Routing**: React Router 7
- **Charts**: Recharts
- **Icons**: Lucide React
- **Forms**: React Hook Form with Zod/HookForm Resolvers

### Backend
- **Language**: Python 3.11+
- **Framework**: FastAPI
- **Database**: SQLAlchemy (SQLite for local, PostgreSQL likely for production)
- **Migrations**: Alembic
- **Deployment**: Railway (Backend), Vercel (Frontend)

## Core Features
1. **Dashboard**: High-level financial health overview.
2. **Income & Expense Tracking**: Categorized transaction management.
3. **Budget Management**: Setting and monitoring monthly limits.
4. **Debt Management & Advisor**: Tracking balances, interest rates, and strategic repayment advice.
5. **Savings Goals**: Tracking progress toward specific financial targets.
6. **Authentication**: Secure user login and registration.
